Welcome to the Simple To-Do Application

This is a full-stack web application built using AngularJS, Node.js, Express.js, and MongoDB. It features secure user authentication (login and registration) and persistent task management. The frontend is developed with AngularJS, while AI tools were used to assist with color selection and minor productivity tasks.

Developed by: Shahid Khan

**Developer Note:** As this is a personal portfolio project, please be aware that the developer, has database access and can view the data you enter.
